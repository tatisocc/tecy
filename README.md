                                                                    B R U T U S  T E X T

Tecy es un conversor de archivos universal diseñado para extraer el "texto bruto" y limpio de diversos formatos de documentos y datos, transformándolo en un dataset simple de palabras, listo para ser consumido por bases de datos o sistemas de procesamiento de lenguaje natural (NLP).

Funcionalidad

    Extracción Robusta: Procesa TXT, CSV, JSON, XML/HTML, XLSX, y PDF.

    Limpieza Agresiva: Elimina activamente números, enlaces (URLs), símbolos y caracteres especiales, manteniendo únicamente palabras alfabéticas (incluyendo caracteres acentuados en español).

    Normalización de Datos: Convierte el texto extraído en una lista de palabras separadas por el delimitador (:), resultando en un formato de texto plano (.txt) uniforme.

Objetivo ("Brutus Text")

El propósito de Tecy es generar el input de texto más puro y desestructurado posible, actuando como el pre-procesador ideal para alimentar sistemas que requieren datos de entrenamiento o diccionarios, como su proyecto Addcy (para diccionarios mejorados).

Tecy garantiza que cualquier fuente de datos se reduzca a la materia prima lingüística necesaria para construir diccionarios y vocabularios de alta calidad.
